## Introduction

Jenesis is a command line tool for rapid contract and service development for the Fetch.ai blockchain ecosystem and other CosmWasm-enabled blockchains ecosystems.

## System Requirements

Jenesis currently requires:

    - **OS**: Linux, MacOS.
    - **Python**: 3.8 to 3.10.
    - **Docker**: 20.10.22 or higher recommended.
    - **Git**: any.
