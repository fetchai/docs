# How to use Fetch Wallet

## Feature 1 - Home Screen

When a user opens the web extension for the first time, they are presented with the account dashboard.

![](src/images/guides/wallet/web-wallet_14.png)

## Account Dashboard Features

### Network Selector Option

By clicking on **Network Selector** button located in the top left corner, the user can change the network.

![](src/images/guides/wallet/web-wallet_15.png)

![](src/images/guides/wallet/web-wallet_16.png)

![](src/images/guides/wallet/web-wallet_17.png)

### Copy Address

By clicking on **Copy** the user can copy their account's address.

![](src/images/guides/wallet/web-wallet_18.png)

### Chat Option

By clicking on this button, the users are able to open the **Chat** window where they can initiate a chat with other users. They can also chat with autonomous agent to ask questions and perform wallet functions.
![](src/images/guides/wallet/web-wallet_19.png)

### Three Dots (More Options)

By clicking on these 3 dots button, you will se the following 3 options to manage your wallet:

![](src/images/guides/wallet/web-wallet_20.png)

### Add New Account

This option allows the user to **Add a new wallet**, as described earlier.

![](src/images/guides/wallet/web-wallet_21.png)

### Change Account

If the user has multiple accounts, they can switch across them using this functionality.

![](src/images/guides/wallet/web-wallet_22.png)

### Rename Account

This function enables users to change the name of their account.

![](src/images/guides/wallet/web-wallet_23.png)

### Delete Wallet

This functionality provides the option to delete a wallet.

![](src/images/guides/wallet/web-wallet_24.png)

![](src/images/guides/wallet/web-wallet_25.png)

**User can also backup their mnemonic seed of account before deleting it by using this function.**

![](src/images/guides/wallet/web-wallet_26.png)

![](src/images/guides/wallet/web-wallet_27.png)

### Viewing Account Information

### Current Balance

The user can see their **Current Balance** in FET (Fetch tokens) and in USD.

![](src/images/guides/wallet/web-wallet_28.png)

## Portfolio

By clicking on the **View Portfolio** button, users can view their portfolio's stats, performance and composition. Here you will have 2 tabs: **Tokens** and **Stats**.

![](src/images/guides/wallet/web-wallet_29.png)

![](src/images/guides/wallet/web-wallet_30.png)

By clicking on **Token** tab, users have the ability to explore the tokens page. Users will be able to see price graph at different timelines but also perform **Receive** or **Send** operations by clicking on token boxes.

![](src/images/guides/wallet/web-wallet_31.png)

On the other hand, the **Stats** tab displays the staking information, including **Available** tokens for staking, **Staked** tokens and **Staking rewards** accrued.

![](src/images/guides/wallet/web-wallet_32.png)

### Portfolio Performance

In the home screen of the Wallet, users can track their portfolio performance over various timeframes (from 24 hours to 1 year).

![](src/images/guides/wallet/web-wallet_33.png)
